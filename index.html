Registration and pre-requisites

This workshop is for everyone interested in containers and have access to any HPC facility. The workshop is very practical, and participants must be familiar with Unix environment and multiple common Unix commands to be able to build and run Singularity containers autonomously. No previous knowledge on containers is necessary.
